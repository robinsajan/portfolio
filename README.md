# How to Add Blog Posts

## Simply add `.txt` files to this folder!

The filename becomes the blog title automatically.

### Example:
- `my-first-blog-post.txt` → displays as "My First Blog Post"
- `understanding-ai-security.txt` → displays as "Understanding AI Security"

### Files needed:
1. **Your blog post files** - Any `.txt` file you add
2. **list.txt** - A simple list of your blog filenames (one per line)

### Example list.txt:
```
my-first-blog-post.txt
understanding-ai-security.txt
learning-pyspark.txt
```

### Blog Post Format:
Just write plain text or simple formatting in your .txt files.

The system will:
- ✅ Use the filename as the title (converts hyphens/underscores to spaces)
- ✅ Use the file's last modified date
- ✅ Sort by newest first
- ✅ Display in Bear Blog style

That's it! No JSON, no complex setup. Just add `.txt` files and update `list.txt`!
